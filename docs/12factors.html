12 Factor App Design

1. Codebase

  • Entire codebase is hosted and stored on GitHub.
  • 3 code repositories: Infra, Backend, and Frontend.
  • All contain only one singular long-living branch for deployment.
  • All edits to code are made through git actions/commits.
  • 3 main deployment stages: Dev, UAT, and Production.
